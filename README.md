Device Tree for Redmi 13R 5G/13C 5G/POCO M6 5G (air)
==========================================

The Redmi 13R 5G/13C 5G/POCO M6 5G (codenamed _"air"_) is a mid-range smartphone from Xiaomi.
It was released in December 2023.

| Basic                   | Spec Sheet                                                                                                                     |
| -----------------------:|:------------------------------------------------------------------------------------------------------------------------------ |
| CPU                     | Octa-core (2x2.2 GHz Cortex-A76 & 6x2.0 GHz Cortex-A55)                                                 |
| Chipset                 | MediaTek MT6835 Dimensity 6100+ (6 nm)                                                                                       |
| GPU                     | Mali-G57 MC2                                                                                                                   |
| Memory                  | 4/6/8 GB RAM                                                                                                                     |
| Shipped Android Version | 13.0                                                                                                                           |
| Storage                 | 128/256 GB UFS 2.2                                                                                                                   |
| Battery                 | Non-removable Li-Po 5000 mAh battery 18W                                                                                           |
| Display                 | IPS LCD 90Hz, 450 nits (typ), 600 nits (HBM)                               |
| Camera (Back)(Main)     | 50 MP, f/1.8, 28mm (wide), PDAF                                                                                 |
| Camera (Front)          | 5 MP                                                                                                                          |
